# PRD

## 产品目标

实现一个企业级 AI 知识库系统，支持用户管理自己的知识库、文档、附件和版本历史，并能通过 AI 完成摘要、问答、标题建议、标签建议和知识库 RAG 问答。

## 用户角色

### 游客

- 注册。
- 登录。
- 查看公开说明页。

### 普通用户

- 管理自己的知识库。
- 管理知识库下的文档。
- 编辑 Markdown。
- 上传附件。
- 查看、回滚和删除文档版本。
- 使用单文档 AI 能力。
- 使用知识库 RAG 问答。

### 管理员

- 管理用户。
- 管理角色。
- 管理权限。
- 为用户分配角色。
- 为角色分配权限。
- 查看基础审计信息。

## 功能边界

第一版实现：

- 认证与当前用户。
- RBAC 与管理员后台。
- 知识库与文档。
- 附件和版本历史。
- Redis 缓存。
- OpenAI 兼容 AI Provider。
- RAG 与 SSE 流式输出。
- Docker Compose 本地部署。

第一版不实现：

- 微服务。
- 网关和注册中心。
- 分布式事务。
- 多租户。
- 多人实时协作。
- 复杂 Agent 工作流。

## 验收主流程

1. 普通用户注册并登录。
2. 创建知识库。
3. 创建 Markdown 文档。
4. 上传附件。
5. 修改文档并生成版本历史。
6. 回滚到历史版本。
7. 删除不再需要的历史版本快照。
8. 生成文档摘要、标题建议、标签建议。
9. 对单文档提问。
10. 对知识库进行 RAG 问答并显示引用。
11. 管理员登录后台，管理用户、角色和权限。
